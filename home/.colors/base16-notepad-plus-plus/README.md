# Base16 for Notepad++

[Notepad++][1] styles based on Chris Kempson's [Base16][2] colour scheme.

## Installation

Download and copy the styles to `%APPDATA%\Notepad++\themes`

## Contributing
Pull requests can be made against [Base16 Builder][3]
